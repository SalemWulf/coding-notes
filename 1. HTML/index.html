HTML Basics

HTML is a markup language and is the basics all websites are made of. This is usually the physical part we see when interacting with a website. HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating web pages. HTML is used to describe the structure of a web page and should generally never be used for styling.

Doctype

At the top of every page there should be a DOCTYPE tag which lets the browser know you're using HTML 5, it should be the first line of code and is generated after typing a ! and hitting enter in VS Code.

Title

The title tag resides in the head of the HTML file and controls the name you see in the browser.

Favicon

To go with a title a favicon is usually needed and is the name of the little image that can be seen in a brower. The syntax is < link> rel="icon" href = "Your Image" < / link> . The favicon goes in the head, usually above the title tag.

Meta

A meta tag often describes your website,the important ones are naturally provided in HTML Boilerplate. Less vital ones contibute to SEO and optimize your website for browser searching.

Comments

You can add comments to an HTML page which make it so that text and tags within the comment aren't picked up by the browser. Use to a minimum. a comment is text enclosed within < !-- â•Œ> . Text should be green or another color to show it's working. These comments can be used to leave notes to yourself or other developers, or even to organize and label code.

HTML Boilerplate

Boilerplate is simply a template for code that makes it easier to write without needing to remember everything. HTML generally has a specific template that goes at the start of every webpage. In VS Code simply type a ! to get it.

Index.html

Index.html should be the file name for the main page on your website that you wish for users to open up first. If it isn't there may be problems when attempting to upload it to the internet, any extra HTML files don't need to follow any naming conventions. Just the main one.

Links

If you wish to link to other files or import frameworks to an HTML file the link tag is generally what you'll use, even favicons use them as well as importing fonts or bootstrap. Links aren't seen in the browser so they go in the head of the HTML file. The syntax is < link link rel="" href="" What goes in the rel and href quotes depends on what you're specifically doing. They also sometimes change places with eachother.

Head

The head is where all the previously discussed tags should go,meta, title, links, they are unseen in the actual webpage as all head elements are. the head section is marked by the head tag

Body

The body is where all the elements the user will see goes such as text and images, this is marked by a body tag. Both the head and body tags are encased in an HTML tag.

HTML Tags

Tag Syntax

Tags in HTML have a syntax and they have to be a certain way in order for the browser browser to recognize them and convert them into elements the user can see and interact with.

Start Tags, End Tags

Most HTML elements have start and end tags. Start tags look like < p>. Two angles enclosing an HTMl element. The end tags look like the same thing with a / coming first. < / p> Everything in between them will be edited to that specific tag. The letter or name within the angles is the element itself, with the angles it's a tag.

Self Closing Tags

Self closing tags don't need an end tag for they don't edit text or contain anything within them. < br> and < hr> are both self closing tags.

br Tag

the br tag is an example of a self closing tag,
it shouldn't be used often
but it seperates text and breaks up lines like it does here.

hr Tag

the hr element stands for horizontal rule and makes a line that reaches across the whole page, can be edited to be more noticable and can be used to split up a webpage or divide sections.

Attributes

All HTML elements can have attributes, these provide additional information about elemenets, they are always specified in the start tag and come with the following syntax. < attribute="value" element>

the href attribute specifies a link and where it goes to

The src attribute is often used with the img element and specifies the path to the image being displayed.

The alt attribute provides alternate text if an image or element doesn't show up, often used for an image tag and has more to do with SEO.

the style attribute can be added to an element to give it styles, it's deprecated though and should never be used in HTML.

the lang attribute is used as an attribute for the html tag and specifies the language, included in boilerplate.

Title Attribute

The title attribute can be added to an element so it's information can be showed when you mouseover an element It is often used for abbreviations or images.

Inline & Block

Block Elements

Block Elements always start on a new line, they always take the full width available. the p tag and div tag are block elements.

Inline elements

inline elements do not start on a new line, they only take up as much width as necessary. img tags, a tags, and button tags are inline.
